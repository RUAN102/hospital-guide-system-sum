医院智能导诊系统 · 产品设计
> 针对患者“不知道挂什么科、线下就医迷路、信息不透明”的痛点，设计的一站式就医服务产品，覆盖导诊、科室/医生查询、院内导航全流程。

---

 📌 项目介绍
- **项目背景**：大医院科室复杂，患者常因不了解症状对应科室、找不到检查室而影响就医体验。
- **核心目标**：通过智能导诊、科室/医生查询、院内导航三大模块，降低用户就医决策成本，提升就医效率。
- **我的角色**：产品负责人，负责用户调研、需求分析、原型设计、PRD撰写与Demo跟进。

---

 ✨ 核心功能模块
| 模块 | 功能说明 | 设计亮点 |
| :--- | :--- | :--- |
| 智能导诊 | 按症状分类引导用户，匹配推荐科室 | 采用“由粗到细”的漏斗式流程，降低用户决策成本 |
| 科室/医生查询 | 支持按科室、姓名、擅长疾病搜索 | 兼顾“有明确目标”和“无明确目标”的用户场景 |
| 院内导航&公告 | 提供院区地图指引与就医须知、公告 | 打通线上预约与线下就诊的服务闭环 |

 📈 项目亮点
1.  **用户导向设计**：通过用户调研，将复杂的医学科室分类转化为“不适部位→症状→科室推荐”的生活化选项，降低用户理解门槛。
2.  **流程防流失设计**：导诊流程进度可视化，减少用户中途放弃率。
3.  **服务闭环设计**：线上导诊预约+线下导航指引，覆盖用户就医全流程。
